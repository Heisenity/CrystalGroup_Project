# Crystal People Lite SOP

## Purpose

Crystal People Lite is a monthly performance review tool for a small internal team.

It helps:

- managers submit structured monthly check-ins
- employees view their own review history
- teams keep all review records in one place

## Who uses it

There are 2 roles:

- `Manager`
- `Employee`

## What the manager can do

- select an employee
- score them in 3 review areas
- write a review comment
- improve the wording before saving
- check whether the wording matches the score
- see who has already been reviewed this month
- see who is still pending
- view team-level insights
- view employee-level growth signals

## What the employee can do

- view the latest review
- see category scores
- read the manager’s written feedback
- track score history
- view personal growth signals and learning actions

## Monthly workflow for managers

### Step 1: Log in as manager

Open the app and sign in with the manager demo account.

### Step 2: Check the month overview

At the top of the manager workspace, review:

- `Reviewed`
- `Pending`
- `Average Score`

These tell you:

- how many employees already have a review this month
- how many are still left
- the overall average score so far

### Step 3: See who is completed and who is still left

Click the reviewed or pending summary card.

A small panel opens with:

- employees already reviewed
- employees still pending

This helps prevent missed check-ins.

### Step 4: Submit a performance review

Go to the `Performance Review` section.

1. Select the employee
2. Score:
   - Output Quality
   - Attendance
   - Teamwork
3. Write a clear comment
4. Click `Refine wording` if needed
5. Review the `Score Fit` section
6. Click `Save check-in`

### Step 5: Review team signals

The `Team Signals` section helps the manager quickly understand the month.

Cards include:

- `Load balance`
- `Wellbeing watch`
- `Check-in watchlist`
- `Team pulse`
- `Recognition notes`
- `Score alignment`

Each card has an `i` icon.

Hover over it to understand what the card means in simple language.

If the card includes `Read more`, click it to open the full explanation.

### Step 6: Use People Lens before a one-to-one

The `People Lens` section helps the manager focus on one employee at a time.

It shows:

- current direction
- average score
- growth summary
- learning actions
- skill signals
- strongest signal

This is useful before:

- one-to-one meetings
- appraisal conversations
- follow-up coaching

### Step 7: Use Recognition Notes

Recognition Notes gives short praise messages for employees doing well.

Use them for:

- Slack appreciation
- email recognition
- meeting shout-outs

## Score meaning guide

- `1` = serious concern
- `2` = below expected level
- `3` = acceptable but needs work
- `4` = strong
- `5` = excellent

## What Score Fit means

`Score Fit` checks whether the written comment matches the actual scores.

Example:

- low scores with very positive wording may need correction
- high scores with very negative wording may also need correction

This helps keep reviews fair and clear.

## What the employee sees

Employees only see their own dashboard.

They do not see the full manager overview.

Their workspace includes:

- `Current Snapshot`
- `Growth Story`
- `Personal Focus`
- `Score History`

## How employees should use it

### Step 1: Log in as employee

Open the app and log in using the employee demo account.

### Step 2: Review the latest snapshot

The first section shows:

- latest review average
- Output Quality
- Attendance
- Teamwork

This gives a quick view of the current month.

### Step 3: Read Growth Story

This section explains recent performance in simple English.

It helps the employee understand:

- what is improving
- what still needs attention

### Step 4: Review Personal Focus

This section highlights:

- strengths
- actions to improve
- skill themes

It can be used as a small monthly action plan.

### Step 5: Review Score History

This section shows all saved monthly check-ins for the employee.

For each entry, the employee can see:

- month
- overall score
- 3 category scores
- written comment

## Where the data is stored

All reviews are saved into Google Sheets.

Each saved row contains:

- timestamp
- month key
- employee email
- employee name
- manager email
- Output Quality score
- Attendance score
- Teamwork score
- written comment

## Best use cases

- monthly employee check-ins
- internal team review tracking
- one-to-one preparation
- recognising good work
- spotting missed reviews before month end

## Good practice for managers

- submit only one review per employee per month if possible
- keep written feedback specific
- include both strengths and next steps
- check pending reviews before the month closes
- use People Lens before difficult conversations

## Good practice for employees

- review feedback monthly
- compare current month to earlier months
- use the action items as a work plan
- raise questions with the manager if feedback is unclear
